# RaidRoll Addon

RaidRoll is an improved World of Warcraft addon designed to facilitate raid loot tracking and rolling. Its streamlined interface and smart features help your team manage loot distribution easily and transparently.

## Features

This add-on integrates essential options for tracking dungeon boss loot, as well as tracking player dice rolls, such as:

- **Fast Loot Tracking:** Track and announce loot drops with clear messages.
- **Simple Dice Roll Window:** Instantly access a dedicated roll window for all loot events.
- **Multi-Profile Support:** Settings are profile-aware, letting you manage configurations for different characters.
- **Improved Compatibility:** Enhanced integration with modern client versions and other popular UI frameworks.

## Improvements

- **Revamped Configuration UI:** The settings panel has been fully rebuilt using the Ace3.0 library for a more intuitive experience. This provides a smoother and more intuitive experience for managing: General Options, Priority Loot, and Loot Window tracking.
- **Minor Bugfixes:** Several minor bugs have been addressed and corrected for better overall stability and performance.
- **Full Spanish Translation:** The addon now includes a complete translation to Spanish, making it more accessible to Spanish-speaking users.

## Slash Commands

- `/rr loot`  
  Opens the Loot Tracker window. Easily view and manage loot drops.
- `/rr show`  
  Displays the dice rolling window, allowing fast rolls for loot distribution.

## Usage

To start using RaidRoll, follow these simple steps:

1.  Download the addon files.
2.  Place the `RaidRoll` folder directly into your World of Warcraft `Interface/AddOns/` directory.
3.  Restart the WoW client or use the command `/reload` in-game to load the interface.
